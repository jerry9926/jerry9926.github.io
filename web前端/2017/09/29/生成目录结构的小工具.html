生成目录结构的小工具

分类：web前端 标签：node工具 发布时间：2017-09-29

npm工具 tree-directory

安装

npm install tree-directory

使用

const tree = require('tree-directory');

tree(__dirname, ['**/*', '!node_modules/**/*']).then(function (res) {
    console.log(res)
});
